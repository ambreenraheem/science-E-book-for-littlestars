# ⭐ Science for Little Stars

An interactive science e-book for kids in Grades 1–3.

## 🚀 How to Deploy on GitHub Pages

### Step 1: Create GitHub Repository
1. Go to https://github.com and log in
2. Click the **+** icon → "New repository"
3. Name it: `science-little-stars` (or any name you like)
4. Set it to **Public**
5. Click "Create repository"

### Step 2: Upload Files
1. Click "uploading an existing file" on the new repo page
2. Drag and drop ALL files and folders:
   - `index.html`
   - `quiz.html`
   - `css/` folder
   - `js/` folder
   - `chapters/` folder
3. Click "Commit changes"

### Step 3: Enable GitHub Pages
1. Go to your repo **Settings**
2. Click **Pages** in the left sidebar
3. Under "Source", select **Deploy from a branch**
4. Branch: **main**, Folder: **/ (root)**
5. Click **Save**

### Step 4: Your site is live! 🎉
After 1–2 minutes, visit:
`https://YOUR-USERNAME.github.io/science-little-stars/`
